## Features

### View components

- [x] Generic button
- [x] Message banner
- [x] Brush (background) view
- [x] Text badge
- [x] Drag indicator

### View modifiers

- [x] Placeholder effect
- [x] Rays effect
- [x] Shine effect

## Requirements

- iOS 15.0+
- Xcode 12.0+
- Swift 5.5+

## Installation

### SwiftPackageManager

```swift
dependencies: [
    .package(url: "https://github.com/macistador/VisualKit", from: "0.0.3")
]
```
